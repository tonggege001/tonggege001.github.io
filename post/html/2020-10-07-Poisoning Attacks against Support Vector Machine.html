数据毒害攻击的基本思路

  1. 找到一个点 xc\boldsymbol{x_{c}}xc​ 加入到训练集中，使验证集的Loss最大。

  2. ∂L∂xc=0              ⟹xt−1=xt+s∗u\frac{\partial L}{\partial \boldsymbol{x_{c}} } = 0 \;\;\;\;\;\;\; \Longrightarrow \boldsymbol{x^{t-1}} = \boldsymbol{x^{t}} + s * \boldsymbol{u}∂xc​∂L​=0⟹xt−1=xt+s∗u，其中 u\boldsymbol{u}u 是关于 xc\boldsymbol{x_{c}}xc​ 的梯度，此步骤是梯度上升法。

  3. 第2步中含有 ∂模型参数∂xc\frac{\partial 模型参数}{\partial \boldsymbol{x_{c}}}∂xc​∂模型参数​ ，很难直接求解出来，所以必须假设模型在学习到 Dorigin∪[(xc,yi)]D_{origin} \cup [(\boldsymbol{x_{c}}, y_{i})]Dorigin​∪[(xc​,yi​)] 后，KKT条件不改变。

对于不等式的优化问题的KKT条件有三个：1. Fermat定理，最优解处梯度为0。 2. 最优解处的点满足约束条件。 3. 在最优点处，对于每个不等式约束必须满足 u * gi(x) = 0。在SVM中使用到了Fermat定理和不等式约束满足条件这两个。

  1. 根据KKT条件不变，可以获得若干个等式，然后对每个等式关于 xc\boldsymbol{x_{c}}xc​ 求导，得到 ∂模型参数∂xc\frac{\partial 模型参数}{\partial \boldsymbol{x_{c}}}∂xc​∂模型参数​ 的表达式，这样就可求解出来。

重要的结论

所有支持向量要么在分类间隔边界上，要么在间隔边界与分类超平面之间，要么在分离超平面误分类一侧。
若 αi⋆<C,ξi=0\alpha_{i}^{\star} < C , \xi_{i} = 0αi⋆​<C,ξi​=0 则支持向量恰好落在间隔边界上。
